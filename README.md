### Development
`git clone the project`

#### Using Docker
`cp evn_sample .env`
`docker compose up -d`

#### Using Nix
you will need postgres DB to connect to
`export the configurations from evn_sample to match your own postgres DB`
`nix develop .`

open http://localhost:4000
